SimpleTrack is a lightweight, zero-configuration web analytics SDK designed for modern web applications. With just 5KB of JavaScript, you get automatic page tracking, custom event monitoring, user analytics, performance metrics, and comprehensive reporting capabilities.

Automatic Tracking

  • Page views, session start, scroll depth, clicks, page exit, performance
  • All events include user/session info, browser data, attribution, and path

Custom Events

  • Track business events, promo campaigns, user engagement, conversions
  • Convenience methods for common patterns (interactions, navigation, errors, performance)

User Management

  • Track authenticated users by email (local only, never sent to server)
  • Complies with basic GDPR requirements

Privacy & Security

  • No cookies by default, anonymous UUIDs, optional email storage
  • All data sent over HTTPS to secure endpoint
